## 🚀 Featured Projects

### ⚖️ Legal Document Intelligence
> Production-grade RAG system for intelligent legal contract analysis

- Built on **510 real SEC contracts** (CUAD dataset) with **80,532 indexed chunks**
- Full Retrieve → Rerank → Generate pipeline: pgvector + Cohere + Groq (Llama 3.1)
- Achieved **Answer Relevancy: 1.000** and **Faithfulness: 0.960** on custom LLM-as-judge evaluation
- Features: Contract Q&A, Clause Extraction (10 types), AI-powered Risk Scoring, Semantic Search
- Deployed on Railway (API) + Streamlit Cloud (UI)

🔗 [View Repository](https://github.com/parth163tar/legal-doc-intelligence) | [Live Demo](https://legaldocintelligence.streamlit.app/)
